# 室内消防烟感项目API接口文档

@[toc]

# 通用说明:

# ①接口通用前缀:

测试环境: http://192.168.0.168:80/frontInterface

生产环境: 待定

# ② 接口通用返回:

{
    "code": "0000", // 正确码为0000 错误码为0001 
    "content": "", // 如发生错误此处会显示相关错误信息
    "result": { // 所有需要的结果集都会放在该对象中
        "id": 194,
        "token": "abb37e1dd94f47289f444b365e932707"
    }
}

# 1 登录模块

# 1.1 各移动端登录通用接口

  • 说明

提供测试的账号 loginType=3&userName=13794532434&passWord=123456

  • 请求URL

/login

  • 请求方式

GET

  • 请求参数
请求参数 必选 参数类型 说明
loginAccount true String 用户名
loginPassword true String 密码
loginType true String APP端类型 1、安装 2、维修 3、业主 4、物业 5、管理
  • 返回字段
返回字段 字段类型 说明
userPic String 用户头像(使用时需自行拼接上前缀)
sex String 用户性别 0男 1女 2未知
name String 用户姓名
phonenumber String 用户手机号
id String 用户ID
email String 用户邮箱
token String 登陆后每次请求都应携带该参数作为登录依据
  • 返回示例
// 正确示例
{
    "content": "",
    "code": "0000",
    "result": {
        "userPic": "2019/05/20/c8e1c9c0c61252861eb933b311aa9816.jpg",
        "sex": 2,
        "name": "王修帅业主",
        "phonenumber": "13794532434",
        "id": 59,
        "email": "qing1212@163.com",
        "token": "fee71211538240d9801a29e4ede58f78"
    },
    "url": null
}
// 错误示例
{
    "content": "当前非维修端账号,禁止登陆!",
    "code": "0001",
    "result": null,
    "url": ""
}

# 2 个人中心模块

# 2.1 修改密码

  • 请求URL

/personalCenter/updatePassWd

  • 请求方式

GET

  • 请求参数
请求参数 必选 参数类型 说明
oldPassWd true String 原密码
newPassWd true String 新密码
  • 返回字段

与通用返回参数一致

  • 返回示例
// 正确示例
{
    "content": "",
    "code": "0000",
    "result": {},
    "url": null
}

{
    "content": "旧密码与系统中的不一致,请检查后重新输入!",
    "code": "0001",
    "result": null,
    "url": ""
}

# 1.2 修改个人信息(姓名、性别、手机号、邮箱)

  • 说明

请求参数为四个 修改哪一属性值 传递哪个即可 非必须全部传递

  • 请求URL

/personalCenter/updateUserInfo

  • 请求方式

GET

  • 请求参数
请求参数 必选 参数类型 说明
userName false String 姓名
sex false String 性别
phone false String 手机号
email false String 邮箱
  • 返回字段

与通用返回参数一致

  • 返回示例
// 正确示例
{
    "content": "",
    "code": "0000",
    "result": {},
    "url": null
}
// 错误示例
{
    "content": "系统错误操作失败!",
    "code": "0001",
    "result": null,
    "url": ""
}

# 4 上传下载模块

# 4.1 上传图片(单张)(例如应用于维修端上传图片上报)

  • 请求URL

http://192.168.0.168:80/common/upload

  • 请求方式

POST

  • 请求参数
请求参数 必选 参数类型 说明
file true File(图片) 上传的图片
  • 返回字段
返回字段 字段类型 说明
msg String 提示消息
code String 返回状态码 0为成功
fileName String 返回文件的绝对路径
url String 返回全路径的url地址
  • 返回示例
// 正确示例
{
    "msg": "操作成功",
    "fileName": "2019/06/28/84ad9748121781ad6f2ebf18814b2903.jpg",
    "code": 0,
    "url": "http://192.168.0.168/profile/upload/2019/06/28/84ad9748121781ad6f2ebf18814b2903.jpg"
}

# 4.2 上传图片(多张)(例如应用于维修端上传图片上报)

  • 请求URL

http://192.168.0.168:80/frontInterface/common/multiUpload

  • 请求方式

POST

  • 请求参数
请求参数 必选 参数类型 说明
files true List 上传的图片多张
  • 返回字段
返回字段 字段类型 说明
listMap List 存储返回图片服务器路径
fileName String 图片相对路径(常用)
url String 返回全路径的url地址
  • 返回示例
// 正确示例
{
    "content": "",
    "code": "0000",
    "result": {
        "listMap": [
            {
                "fileName": "2019/07/04\\524a9eb57981e6f0a1e52ae48257d363.jpg",
                "url": "http://192.168.0.168/profile/upload/2019/07/04\\524a9eb57981e6f0a1e52ae48257d363.jpg"
            },
            {
                "fileName": "2019/07/04\\66b0349505dc97b16ed3fa4a10f5853a.png",
                "url": "http://192.168.0.168/profile/upload/2019/07/04\\66b0349505dc97b16ed3fa4a10f5853a.png"
            }
        ]
    },
    "url": null
}
// 错误示例
{
    "content": "图片上传失败,请稍后重试!",
    "code": "0001",
    "result": null,
    "url": ""
}
