# 1.1 工单列表查看(消防 烟感 水压)

  • 说明
  • 请求URL

/repairerFront/getEquipmentFailurelist

  • 请求方式

POST

  • 请求参数
请求参数 必选 参数类型 说明
equipmentType true Integer 设备类型 筛选条件 0为消防栓设备 1为烟感器设备 2为水压设备
alarmReason false Integer 报警原因 筛选条件
消火栓:110=用水(异常)；120=开盖(异常);130=电池电压低(故障)；200=离线(故障);
烟感器:130=电池电压低(故障)；200=离线(故障);
消防水压:140=水压低(异常);130=电池电压低(故障)；200=离线(故障);
alarmStatus false Integer 处理状态 筛选条件 1未处理 3已处理
fillDate false String 报警日期 筛选条件
pageNum true Integer 当前页 筛选条件
pageSize true Integer 页大小 筛选条件
  • 返回字段
返回字段 字段类型 说明
address String 地址
latitude String 纬度
longitude String 经度
alarmDate String 报警日期
deviceId Integer 设备ID
equipmentType Integer 报警类型(0消火防1烟感器 2消防水压)
personName String 维修人(登录人)名称
alarmStatus Integer 处理状态 1未处理 3已处理
person Integer 维修人(登录人)ID
faId Integer 报警事件ID
equipmentName String 设备名称
repairDatetime String 处理时间
alarmReason String 报警原因 筛选条件 130为电池电压低 200为离线
  • 返回示例
